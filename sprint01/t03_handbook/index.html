HTML

p

If we want a line to appear as such, we can define it as a "paragraph" by enclosing it in paragraph element tags.
<p>This is a paragraph.</p>

This is a paragraph.

h1

The "h1" element is used to define the most important heading on the page.
<h1>This is a...</h1>

This is a...

b

The "b" element is used to bold text.
<b>Text</b>

Text

ul

The "ul" element is used to create an unordered list. Unordered lists are used for items for which the order doesn't matter.
<ul>
<li>Cat</li>
<li>Dog</li>
</ul>

  • Cat
  • Dog

ol

The "ol" element is used to create an ordered list. Ordered lists are used for items that should be in a specific order.
<ol>
<li>Cat</li>
<li>Dog</li>
</ol>

  1. Cat
  2. Dog

table

The "table" element is used to create a table.
<table>...</table>

PROTOCOLS

Description

Protocols are a set of rules that determine how data is transmitted between devices.

HTTP/HTTPS

Methods:
GET requests a representation of the specified resource.
POST submits data to be processed to a specified resource.
PUT uploads a representation of the specified resource.
DELETE deletes the specified resource.
PATCH is used to apply partial modifications to a resource.

Status code
1xx: Informational
example: 100 Continue
2xx: Success
example: 201 Created
3xx: Redirection
example: 301 Moved Permanently
4xx: Client Error
example: 404 Not Found
5xx: Server Error
example: 500 Internal Server Error

DNS

A - Address record. It is used to map domain names to IP addresses.
AAAA - IPv6 address record. It maps a domain name to an IPv6 address.
CNAME - Canonical name record. It is used to alias one name to another.
HINFO - Host information record. It is used to provide information about the host.
ISDN - ISDN address record. It is used to provide ISDN information.
MX - Mail exchange record. It is used in routing email.
NS - Name server record. It delegates a DNS zone to use the given authoritative name servers.
PTR - Pointer record. It is used for reverse DNS lookups.
SOA - Start of authority record. It specifies authoritative information about a DNS zone.
TXT - Text record. It is used to store text-based information.
